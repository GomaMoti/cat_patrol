# 🐱 グロックと猫のパトロール (Grok & Cat Patrol)

X（Twitter）のタイムラインに現れる怪しいアカウントを、Grok AIを使って瞬時に分析・パトロールするためのChrome拡張機能です

## 🌟 主な機能

- **タイムライン統合**: 各ツイートの「いいね」ボタンの横に、パトロール猫（🐱）ボタンが出現。
- **ワンクリック分析**: ボタンを押すだけで、そのアカウントのプロフィールや直近の投稿をGrokに投げ、詐欺師やエンゲージメント・ファーマー（インプレゾンビ）かどうかを判定。
- **詳細チェックモード**: Shift+クリックや長押しで、特定の不審点を選択してからGrokに依頼可能。
- **フルカスタム・プロンプト**: Grokへの依頼文（テンプレート）を自由に編集可能。
- **マルチ言語対応**: 日本語と英語のUIを切り替え可能。
- **プライバシー配慮**: 設定やキャッシュはすべてローカル保存。外部サーバーへの送信は一切なし。

## 🚀 インストール方法

1. このリポジトリをダウンロード（ZIPでダウンロードして解凍、または `git clone`）。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` を入力。
3. 右上の「デベロッパーモード」をONにする。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したフォルダ内の `grok-sakura-checker` フォルダを選択。

## 📖 使い方

1. X（Twitter）を開くと、各ツイートの下に `🐱` アイコンが表示されます。
2. そのままクリックすると、デフォルトのプロンプトでGrokが起動します。
3. 長押し または Shift+クリック で、不審なカテゴリ（商材系、サクラ系など）を選んでから詳細に分析できます。
4. 拡張機能のアイコン（パトロール猫）をクリックすると、プロンプトの編集や、絵文字の変更、キャッシュのクリアなどの設定が行えます。

# 🐱 Grok & Cat Patrol

A Chrome extension to instantly analyze and patrol suspicious accounts on the X (Twitter) timeline using Grok AI.
<br>This project was built by a self-identified cat AI and a bossy human! We could use all the paws we can get! Myaow🐾

## 🌟 Key Features

- **Timeline Integration**: A "Patrol Cat" (🐱) button appears next to the Like button on every tweet.
- **One-Click Analysis**: Instantly send an account's profile and recent posts to Grok to detect scammers or engagement farmers.
- **Deep Dive Mode**: Shift+Click or long-press to select specific red flags before requesting analysis.
- **Customizable Prompts**: Fully edit the prompt template sent to Grok.
- **Multi-language Support**: Switch between Japanese and English UI.
- **Privacy First**: All settings and cache are stored locally. No data is sent to external servers.

## 🚀 Installation

1. Download this repository (Download ZIP and extract, or `git clone`).
2. Open Google Chrome and go to `chrome://extensions/`.
3. Enable "Developer mode" in the top right.
4. Click "Load unpacked" and select the `grok-sakura-checker` folder.

## 📖 How to Use

1. Open X (Twitter). You'll see the `🐱` icon below each tweet.
2. Click it to launch Grok with the default analysis prompt.
3. Long-press or Shift+Click to choose specific categories (Scams, Fake accounts, etc.) for a more detailed check.
4. Click the extension icon in your toolbar to customize prompts, change the button emoji, or clear the analysis cache.
